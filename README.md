# Editor de PDF

Aplicação em Bun + Elysia + Svelte 5 para preencher a procuração em PDF, baixar o arquivo preenchido e acessar o assinador gov.br.

## Instalação

```sh
bun install
```

## Desenvolvimento

Em um terminal, rode o backend:

```sh
bun run dev
```

Em outro terminal, rode o frontend:

```sh
bun run dev:web
```

## Produção

```sh
bun run build
bun run start
```

Depois do build, acesse `http://localhost:3000`.
